# Laporan Praktikum

<img width="1920" height="1008" alt="Image" src="https://github.com/user-attachments/assets/bc2300e2-4ab1-4d71-ad07-17848c3deb59" />

- Tahap pertama adalah membuat struktur dasar dokumen HTML yang terdiri dari bagian kepala (head)     dan isi (body). Struktur ini menjadi kerangka utama semua halaman praktikum yang akan dibuat.


<img width="960" height="1008" alt="Image" src="https://github.com/user-attachments/assets/c6223597-f706-40ee-acac-fb5501feca43" />

- Pada tahap ini ditambahkan ordered list untuk menampilkan daftar item secara berurutan. Ordered     list menampilkan data dalam bentuk urutan angka, huruf, atau angka romawi tergantung pada atribut   yang digunakan. Tujuannya adalah menampilkan data yang memiliki urutan tertentu, seperti langkah    kerja atau peringkat.

<img width="960" height="1008" alt="Image" src="https://github.com/user-attachments/assets/47ed8d16-154d-4868-8fe7-979b02efd2c7" />

- Unordered list digunakan untuk menampilkan daftar tanpa urutan. Setiap item diberi simbol seperti   titik atau kotak. Atribut list dapat diubah untuk mengganti bentuk simbol, misalnya disc, circle,   atau square. Daftar ini cocok digunakan untuk menampilkan item yang setara, seperti komponen atau   fitur.


<img width="960" height="1008" alt="Image" src="https://github.com/user-attachments/assets/12c3ffe8-65b1-4a86-88d1-1e2cc20554df" />

- Description list terdiri dari istilah dan penjelasan. Elemen ini berguna untuk mendeskripsikan      pasangan data seperti nama beserta arti atau definisi dari istilah tertentu.


<img width="960" height="1008" alt="Image" src="https://github.com/user-attachments/assets/388f67c9-9e12-45cf-a4e5-63208e290224" />

- Pada bagian ini dilakukan percobaan membuat list di dalam list. Nested list menampilkan struktur    data yang memiliki sub-item, misalnya kategori utama dan subkategori. Dengan cara ini, hubungan     antar data dapat divisualisasikan secara hierarkis.


<img width="960" height="1008" alt="Image" src="https://github.com/user-attachments/assets/7cbfa154-ab36-452f-985e-4d9086e93413" />

- Selanjutnya, dilakukan pembuatan tabel untuk menampilkan data dalam bentuk baris dan kolom. Tabel   digunakan agar data lebih mudah dibaca dan disusun dengan rapi. Tabel terdiri dari bagian kepala    tabel (thead) dan isi tabel (tbody).


<img width="960" height="1008" alt="Image" src="https://github.com/user-attachments/assets/493d7da1-2f67-40be-b06f-eaf9668e7268" />

- Pada tabel juga diterapkan penggabungan sel menggunakan atribut rowspan dan colspan. Atribut        colspan digunakan untuk menggabungkan beberapa kolom, sedangkan rowspan untuk menggabungkan         beberapa baris. Hal ini berguna untuk membuat tampilan tabel yang lebih kompleks, seperti tabel     nilai atau data laporan



<img width="960" height="1008" alt="Image" src="https://github.com/user-attachments/assets/f6d54166-e9c2-48bd-b6da-4252cc3c4d19" />

- Tahap berikutnya adalah membuat form untuk menerima input dari pengguna. Form terdiri dari          beberapa elemen seperti input teks, area teks, radio button, dan tombol submit. Form berfungsi      sebagai media komunikasi antara pengguna dan sistem.


<img width="960" height="1008" alt="Image" src="https://github.com/user-attachments/assets/fd94ec2c-4692-4df9-b2f4-1f9008e6dafc" />

- CSS ditambahkan untuk mempercantik tampilan form. Dengan CSS, tampilan setiap elemen dapat diatur   seperti warna border, jarak antar elemen, serta tampilan tombol. Hal ini membuat form lebih         menarik dan mudah digunakan
